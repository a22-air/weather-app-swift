# 天気アプリ（Weather App）

Swiftで作成したシンプルな天気アプリです。現在地の天気をAPIから取得して表示します。

## 主な機能
- 都道府県の天気取得
- 天気アイコンの表示
- エラー処理（都道府県以外の項目が押下された場合）

## 技術スタック
- Swift
- Storyboard（AutoLayout使用）
- OpenWeatherMap API
